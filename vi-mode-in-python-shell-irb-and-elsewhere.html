Vi Mode in Python Shell, IRB, and Elsewhere!

by Tim Henderson
Tue 04 February 2014

Enable vi mode in irb, python shell, and other readline based systems:

cat <<END >>~/.inputrc
> set -o vi
> set editing-mode vi
> END

Now everything has vi key bindings! You are welcome.
